### 🔥 Currently Building

**Crypto Prediction Market Trading Engine**

A systematic trading engine for 5-minute directional prediction markets on cryptocurrency pairs.

| Component | Description |
|-----------|-------------|
| **Signal Generation** | Order flow imbalance, microstructure signals, cross-asset momentum (BTC, ETH, SOL) |
| **Models** | Ensemble of gradient-boosted models + LSTMs with calibrated confidence intervals |
| **Execution** | Low-latency OMS with Kelly-based position sizing and real-time PnL tracking |
| **Risk** | Dynamic exposure limits, drawdown controls, correlation-aware constraints |

`Status:` Live paper trading · 63% directional accuracy on 5-min BTC/USDT · Sharpe > 1.8 on backtests
